# Customer Sentiment BI Automation (Power BI + Python)

## 📊 Project Overview
An advanced business intelligence dashboard that bridges data engineering automation with interactive text analytics. This project leverages Python to perform automated, rule-based text sentiment processing on raw unstructured customer reviews and seamlessly integrates the computed scoring metrics into Power BI for behavioral trend tracking and executive visualization.

## ⚙️ Core Technical Workflow
* **Python Text Processing Engine:** Builds data pipelines to clean raw feedbacks, applying text analysis logic to dynamically score phrases into positive, neutral, or negative categories.
* **Data Integration:** Exports the automated Python metrics directly into structured tables optimized for Power BI Desktop.
* **Operational Dashboard:** Creates visualization interfaces in Power BI to monitor dynamic product categories, track customer satisfaction spikes, and analyze metric distribution.

## 🧰 Tech Stack
* **Language & Automation:** Python (Pandas, NumPy, Rule-Based Natural Language Logic)
* **Business Intelligence & ETL:** Power BI Desktop, Power Query
* **Environment:** Jupyter Notebook / Google Colab
